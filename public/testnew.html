CoReg Widget Test - New Version

This page demonstrates the updated questionnaire widget with CashApp landing page design.

New Widget Features

  • CashApp $1000 gift card landing page
  • Green gradient background design
  • Customer reviews section
  • Full questionnaire flow
  • Email capture step
  • Targeted ad serving

Widget Implementation

The widget is loaded using this script tag:

<script src="/widget.js"></script>

Note: The widget should appear as a full-page overlay starting with the CashApp offer landing page.

How it works:

  1. Widget loads automatically with CashApp landing page
  2. User sees $1000 gift card offer with reviews
  3. User clicks Yes/No to start questionnaire
  4. Questions are fetched from your admin panel
  5. After questions, user enters email to "claim gift card"
  6. Targeted ads are served based on responses
  7. Analytics data is collected for reporting
