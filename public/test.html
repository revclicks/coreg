How it works:

  1. The widget loads automatically when the page loads
  2. It fetches questions from your admin panel
  3. Users answer the questions in sequence
  4. After completion, a targeted ad is shown based on their responses
  5. All interactions are tracked for analytics

Widget Implementation

The widget is loaded using this script tag:

<script src="https://cdn.coreg.com/sites/NqQ8M6AAYo.js"></script>

Note: The widget should appear as a full-page overlay starting with the CashApp offer landing page.
